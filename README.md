# Graphqyl

## Dependencies

* node
* npm

## Getting Started

1. `npm install`
2. `npm start`

## Docker Image

1. `docker -t <IMAGENAME> build .` to build and tag the image
2. `docker -P run <IMAGENAME>` to run the image and map a random host port to the exposed container port (4000)
